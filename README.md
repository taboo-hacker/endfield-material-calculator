## 项目介绍

终末地物资调度计算器是一款专为游戏《终末地》设计的物资收益分析工具，帮助玩家优化物资调度策略，最大化收益。

## 功能特点

- **多地区支持**：支持四号谷地和武陵两个地区的物资收益计算
- **实时计算**：输入价格后自动计算收益数据，提供实时反馈
- **智能分析**：生成专业的物资调度分析报告，推荐最优采购策略
- **数据可视化**：通过图表直观展示收益数据，包括总利润和收益率
- **排序功能**：支持按收益率或总利润排序，方便快速找到最优物资
- **批量操作**：提供一键填充默认价格和清空数据功能
- **离线使用**：所有资源已本地化，无需网络连接即可使用

## 技术栈

- HTML5
- Tailwind CSS
- JavaScript
- Chart.js

## 使用方法

1. **下载项目**：将项目下载到本地目录
2. **打开应用**：直接双击 `endfield-material-calculator.html` 文件，或通过本地服务器访问
3. **选择地区**：在顶部导航栏选择要分析的地区（四号谷地或武陵）
4. **输入价格**：在物资录入区输入每种物资的实际购入价和售出价
5. **查看结果**：系统会自动计算收益数据，并在结果区显示分析报告和图表
6. **优化策略**：根据分析报告调整采购策略，最大化收益
